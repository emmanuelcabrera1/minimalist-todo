# Documentation

## Intent
Contains meta prompts, security protocols, and documentation management systems for the workspace.

## Contents
| File | Purpose |
|------|---------|
| `GITHUB_DOC_MANAGER.md` | Secure GitHub documentation management meta prompt |
| `audits/UXUI_AUDIT.md` | Latest UX/UI audit report |
| `protocols/SECURITY_PROTOCOL.md` | Security guidelines and protocols |
| `templates/AUDIT_PROMPT_TEMPLATE.md` | Template for conducting audits |

## Security Protocols
All documentation operations follow the GITHUB_DOC_MANAGER security protocols:
- Repository whitelisting
- Input validation
- Sensitive data scanning
- Protected branch enforcement

## Usage
Access via `/github-docs` workflow or directly read the meta prompt.
