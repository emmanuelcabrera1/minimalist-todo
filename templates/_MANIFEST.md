# Templates

## Intent
Starter templates for creating new projects that comply with workspace design standards.

## Available Templates
| Template | Usage |
|----------|-------|
| `new-project-template/` | Base HTML/CSS project with design token structure |
| `standard-pwa-template/` | **RECOMMENDED**. Implements Zero-Entropy standards (Docs, Tests, Structure). |

## Usage
```bash
cp -r templates/new-project-template/ projects/my-new-project/
```

Then update the project's `_MANIFEST.md` and register in `projects/_MANIFEST.md`.
